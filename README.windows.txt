Последовательность установки прошивки в МВ77.07 в OS Windows
------------------------------------------------------------

ПРЕДУПРЕЖДЕНИЕ: После выполнения данной процедуры все данные в NAND памяти
                микрокомпьютера будут потеряны! Перед записью прошивки,
		          убедитесь, что у Вас есть резервная копия!


0. Установите пакет ПО WinPCAP.
1. Отключите питание микрокомпьютера.
2. Наденьте перемычку boot
3. Соедините микрокомпьютер ethernet кабелем с хост компьютером
4. Подключите питание микрокомпьютера
5. Запустите edcl сценарий eupgrade.edcl
   Это можно сделать из командной строки командой

   edcltool -f имя_edcl_сценария.edcl -i N

   где N - номер сетевого интерфейса. По умолчанию используется 0.
   Список сетевых интерфейсов можно посмотреть запустив edcltool -l
   Если у компьютера только одна сетевая карта - для выполнения этого
   шага можно использовать сценарий flash.bat

   В данном комплекте поставки есть следующие сценарии edcl:

   * eupgrade-serialcon.edcl - Прошивка будет использовать терминал
     на последовательном порту для вывода журнала загрузки ядра.

   * eupgrade-hdmicon.edcl - Прошивка будет использовать графический терминал
     на экране подключенном через HDMI для вывода журнала загрузки ядра.

Пример:
   edcltool -f eupgrade-serialcon.edcl -i 0


6. Дождитесь окончания процесса записи прошивки в микрокомпьютер.
7. Отключите питание, снимите перемычку boot.
8. Поздравляем, процесс прошивки завершен, можно подать питание и работать.

Параметры по умолчанию
-----------------------
IP: 192.168.20.9
NETMASK: 255.255.255.0
Login: root
Password: 12345678
Serial: 38400 8n1
