Последовательность установки прошивки в МВ77.07
------------------------------------------------

ПРЕДУПРЕЖДЕНИЕ: После выполнения данной процедуры все данные в NAND памяти
                микрокомпьютера будут потеряны! Перед записью прошивки, 
		убедитесьчто у Вас есть резерная копия!

0. Соберите и установите в систему ПО edcltool из набора программного 
   обеспечения для МВ77.07. Инкструкции по установке можно найти в архиве с
   edcltool.  
1. Отключите питание микрокомпьютера. 
2. Наденьте перемычку boot
3. Соедините микрокомпюьтер ethernet кабелем с хост компьютером
4. Подключите питание микрокомпьютера
5. "Поднимите" интерфейс, к которому подключен микрокомпьютер. 
   Назначать IP адрес не требуется.
   Это можно сделать, например, командой 
       sudo ifconfig eth0 up
   Вместо eth0 используйте имя интерфейса, к которому подключен микрокомпьютер.  
5. Запустите подходящий edcl сценарий 
   Это можно сделать командой 

   edcltool -f имя_файла_сценария.edcl -i ethX 

   где ethX - Имя сетевого интерфейса. По умолчанию используется eth0. 

   В данном комплекте поставки есть следующие сценарии edcl: 

   * eupgrade-serialcon.edcl - Прошивка будет использовать терминал 
     на последовательном порту для вывода журнала загрузки ядра.

   * eupgrade-hdmicon.edcl - Прошивка будет использовать графический терминал 
     на экране подключенном через HDMI для вывода журнала загрузки ядра.

Пример: 
   edcltool -f eupgrade-serialcon.edcl -i eth0

6. Дождитесь окончания процесса записи прошивки в микрокомпьютер. 
7. Отключите питание, снимите перемычку boot. 
8. Поздравляем, процесс прошивки завершен, можно подать питание и работать.
